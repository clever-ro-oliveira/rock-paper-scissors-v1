# Rock Paper Scissors - PHP

Mini projeto desenvolvido em PHP puro para demonstrar:

- POO
- API JSON
- AJAX
- Persistência de dados
- Arquitetura organizada

## Tecnologias

- PHP
- JavaScript
- Fetch API
- JSON storage

## Funcionalidades

✔ Jogar contra o computador  
✔ Placar persistente  
✔ Reset do placar  
✔ API REST simples

## API Endpoints

POST /api.php?action=play

GET /api.php?action=score

POST /api.php?action=reset


## Executar

```bash
php -S localhost:8000
```
